# EEG-EMG Fusion for Upper-Limb Movement Classification

**Author:** Muskaan Garg | H00416442 | Heriot-Watt University
**Dataset:** [NeBULA — Garro et al., Nature Scientific Data (2025)](https://doi.org/10.1038/s41597-025-05042-4)  
**Dataset files:** [figshare](https://doi.org/10.6084/m9.figshare.27301629)

---

## Overview

This project implements and evaluates a complete EEG-EMG deep learning fusion pipeline for subject-independent upper-limb reaching classification on the NeBULA dataset. Five models are trained and compared: EEGNet, EEG CNN-LSTM, EMGNet, EMG CNN-LSTM, and a gated fusion model. A systematic EEG diagnostic pipeline and a trial-level EEG-to-EMG timing analysis are also included.

All analyses use the free (unassisted) condition only. Three subjects were excluded prior to analysis (sub-03, sub-12, sub-21), leaving 37 usable subjects.

## Usage

### Step 1 — Preprocessing

Place the raw NeBULA BIDS data in `data/`. Then run:

```bash
python epoch.py
```

This produces windowed arrays in `preprocessed/` and per-subject trial arrays in `preprocessed/sub-XX/`.

### Step 2 — Diagnostics (optional but recommended)

```bash
python eda/feasibility.py
python eda/eeg_diagnostics.py
python eda/emg_diagnostics.py
```

### Step 3 — Train subject-independent models

```bash
python models_sub_independant/emg_net.py
python models_sub_independant/emg_cnn_lstm.py
python models_sub_independant/eeg_net.py
python models_sub_independant/eeg_cnn_lstm.py
python models_sub_independant/fusion.py
```

Results are saved to `models_sub_independant/results/`.

### Step 4 — Subject-dependent cross-check (optional)

```bash
python subject_dependant/epoch_sub_dependant.py
python subject_dependant/models_sub_dep/eeg_net_dep.py
python subject_dependant/models_sub_dep/emg_net_dep.py
```

### Step 5 — Timing analysis

```bash
python temporal_analysis/temporal_pipeline.py
```

Results are saved to `temporal_analysis/timing_results/`.

## Results (subject-independent, seed=42)

Test subjects: 8, 9, 17, 24, 30, 33

| Model | Modality | Accuracy | Macro F1 |
|---|---|---|---|
| EEGNet | EEG | 33.9% | 0.305 |
| EEG CNN-LSTM | EEG | 35.2% | 0.337 |
| EMGNet | EMG | 79.9% | 0.801 |
| EMG CNN-LSTM | EMG | 84.1% | 0.842 |
| Fusion | EEG + EMG | 78.2% | 0.784 |
| Chance baseline | — | 33.3% | 0.333 |

---

## Key Findings

- EMG CNN-LSTM achieved the strongest result at 84.1% macro F1
- Both EEG models remained at chance regardless of architecture
- Ten independent diagnostic analyses confirmed EEG task-discriminability failure for these reaching variants
- The fusion model did not outperform EMG-only — the learned gate suppressed EEG contribution throughout training
- Mean EEG-to-EMG delay: ~180ms across all three tasks (36 subjects, free condition)
